# Documentation

This directory contains detailed documentation for the NeuroPulse project.

## Files

### `architecture-overview.md`
**Technical architecture documentation.**

- System architecture diagrams
- Component interactions
- Data flow diagrams
- Technology stack details
- Design decisions and rationale

**Use this for:**
- Understanding how the system works
- Technical deep-dives
- Architecture reviews

### `dashboard-layout.md`
**Dashboard UI/UX documentation.**

- Dashboard layout and panels
- Component descriptions
- User interface flow
- Visual design guidelines

**Use this for:**
- Understanding the dashboard structure
- UI/UX design decisions
- Frontend development

### `demo-storyboard.md`
**Demo script and storyboard.**

- Step-by-step demo flow
- Key features to highlight
- Talking points for presentations
- Demo checklist

**Use this for:**
- Preparing hackathon presentations
- Demo walkthroughs
- Feature highlights
